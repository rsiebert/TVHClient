Information
TVHClient beinhaltet viele kostenlose Funktionen. Dies wird auch in Zukunft so bleiben. Für einen kleinen Betrag können aber zusätzliche Funktionen freigeschaltet werden. Durch den Kauf zeigst Du Deine Unterstützung und förderst die weitere Entwicklung. Mit dem Kaufen Button erscheint ein neuer Dialog in dem Du den Kauf bestätigen mußt. Nach Abschluss des Bezahlvorganges kann das Fenster geschlossen werden. Es sind anschließend alle Funktionen freigeschaltet.
Biite beachte, dass eine Funktion nur dann aktiviert wird, wenn der Server diese unterstützt. Im Status Bildschirm muss mindestens die angegebene API-Version angezeigt werden!
Vielen Dank!
Preis
Der Freischalter kostet in Deutschland 1,99€. In anderen Ländern kann der Preis etwas variieren, da dort andere Mehrwertsteuersätze gelten.
Zusätzliche Funktionen
  • Einmal-Aufnahmen hinzufügen
    Falls die Programübersicht das gewünschte Programm nicht anzeigt, kannst Du die Aufnahme manuell erstellen. Diese wird dann nur einmal aufgenommen.
  • Aufnahmen bearbeiten
    Bearbeite nachträglich Programme die geplant sind oder schon aufgenommen werden. Titel, Beschreibung und andere Parameter wie z.B. Start- und Stopzeit können verändert werden.
  • Manuelle Aufnahmen
    Bestimme selbst, welche Programme wann und wie aufgenommen werden. Definiere diese Aufnahmen durch Parameter wie z.B. Start- und Endzeit, Tag der Aufnahme und mehr. (Mindestens API-Version 18)
  • Mehr Kontrolle über Serienaufnahmen
    Erstelle exaktere Serienaufnahmen mit Hilfe zusätzlicher Parameter wie z.B. Start- und Endzeit, Dauer, Tag der Aufnahme und mehr. Ausserdem können bestehende Serienaufnahmen editiert werden
    (Mindestens API-Version 13)
  • Streaming Profile
    Diese sind die neueren Wiedergabe- und Aufnahmeprofile, die vom Server bereit gestellt werden.
    (Mindestens API-Version 16)
  • Suchen während der Eingabe
    Im Suchbildschirm startet die Suche wenn mindestens 3 Zeichen eingegeben werden.
  • Aufnahmen herunterladen
    Aufnahmen können direkt auf das Handy oder Tablet heruntergeladen werden, um sie später zu schauen oder mit Freunden zu teilen.
  • Server aufwecken
    Verwende Wake On LAN (WOL) um einen schlafenden oder ausgeschalteten TVHeadend-Server durch TVHClient zu starten.
