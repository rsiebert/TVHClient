Information
TVHClient contains a variety of features which will remain free in the future. Additional features can be enabled for a small fee. This ensures the further development and shows your support. If you select the purchase menu a new dialog will open where you must confirm the purchase. Once the purchase is done the dialog can be closed and all features will be available.
Thank you!
Additional Features
  • Streaming profiles
    These are the newer playback and recording profiles that are provided by the server.
    (Your server must support this feature. Check the status page that your servers API version is at least 16.)
  • More control over series recordings
    Fine tune your series recordings! Define more customized series recordings by specifying additional parameters like start and stop time, duration, day of week to record and more. Additionally you can edit existing series recordings.
    (Your server must support this feature. Check the status page that your servers API version is at least 13.)
  • Support for timer recordings
    Manually define when and how programs are recorded. Specify additional parameters like start and stop time, day of week to record and more.
    (Your server must support this feature. Check the status page that your servers API version is at least 17.)
  • Search as you type
    In the search screen the search will start automatically when 3 or more characters are entered. The search results will automatically be updated whenever the search string changes.
  • Edit recordings
    Fine tune your scheduled or already recording programs! Edit additional parameters like start and stop time, pre- and postrecording times, and more.
