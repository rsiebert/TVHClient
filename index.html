TVHClient

An Android client for TVHeadend using the HTSP protocol

Download this project as a .zip file Download this project as a tar.gz file

TVHClient is an Android client for TVHeadend, a TV streaming server. Some of its featrues are:

  • List all channels and their current programs
  • List all recordings
  • Show a complete and customizable program guide
  • Search the EPG for upcoming programs
  • Schedule, cancel and remove recordings
  • Stream audio and video
  • Show status information
  • Connect to multiple servers
  • Separate playback profiles for programs and recordings
  • Manually select a different language
