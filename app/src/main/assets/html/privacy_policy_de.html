Allgemeines
TVHClient wird entwickelt von Robert Siebert.

Mit TVHClient können Sie mit TVHeadend (einem TV-Streaming-Server und -Recorder) interagieren, um Ihre TV-Programme auf Ihrem Mobiltelefon oder Tablet anzusehen, aufzuzeichnen und zu verwalten.

TVHClient wird kostenlos zur Verfügung gestellt und ist so wie es ist zu verwenden. Diese Seite informiert Sie über meine Richtlinien bezüglich der Erfassung, Verwendung und Offenlegung persönlicher Informationen. Wenn Sie sich für die Verwendung von TVHClient entscheiden, stimmen Sie der Erfassung und Verwendung von Informationen im Zusammenhang mit dieser Richtlinie zu. Die von mir erfassten persönlichen Daten werden zur Bereitstellung und Verbesserung der Software verwendet. Ich werde Ihre Informationen nur wie in dieser Datenschutzrichtlinie beschrieben verwenden oder an Dritte weitergeben.

Informationserfassung und -nutzung
TVHClient speichert Benutzereinstellungen und die vom Server empfangenen Daten im geschützten Speicher Ihres Android-Geräts. Die Anmeldeinformationen werden nur zum Herstellen einer Verbindung mit dem konfigurierten Server verwendet. Die vom Server empfangenen Daten werden nur von der App verwendet, um sie auf dem Bildschirm anzuzeigen.

Für eine bessere Erfahrung bei der Nutzung meiner App kann es sein, dass Sie uns bestimmte personenbezogene Daten zur Verfügung stellen müssen. Diese Informationen werden als Protokolldaten auf Ihrem Telefon gespeichert. Diese Daten können Informationen wie die IP-Adresse Ihres Geräts, den Gerätenamen, die Betriebssystemversion, die Version der App, die Uhrzeit und das Datum Ihrer Nutzung der App sowie andere Statistiken enthalten. Die von mir angeforderten Informationen werden auf Ihrem Gerät gespeichert und in keiner Weise von mir erfasst.

Darüber hinaus verwendet die App Dienste von Drittanbietern, die möglicherweise Informationen sammeln, mit denen Sie identifiziert werden können.

Berechtigungen
Der Zugriff auf den Gerätespeicher ist erforderlich, um Aufzeichnungen herunterzuladen und Log-Dateien zu speichern. Der Zugriff wird ausschließlich für diese Funktionen benötigt und nur wenn von Ihnen aktiviert. Die Anwendung funktioniert auch ohne die Berechtigung. Sie können den Zugriff auf den Gerätespeicher ab Android 6 einschränken.
